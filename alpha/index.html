Triggers

  • Hover off "Main Level" button on outlet card

Rules

Hover state on:
  • Cursor changes to pointer
  • Dark gray gradually fades in to replace light gray fill of button
Hover state off:
  • Cursor reverts to default
  • Dark gray fill of button gradually fades out to return button to light gray fill

Alpha Visualization

Alpha Overview - New Design

Triggers

  • Hover over "Main Level" button on outlet card
  • Hover off "Main Level" button on outlet card

Rules

Hover state on:
  • Cursor changes to pointer
  • Black slides in to fill the button from left to right; 0.3 seconds
  • Location icon and location text move to the left by 8 pixels; 0.3 seconds
  • Arrow symbol fades in from 0% opacity to 100% opacity as it slides in from left to right by 8 pixels; 0.3 seconds
Hover state off:
  • Cursor reverts to default
  • Black fill slides from right to left to reveal dark gray fill of button; 0.3 seconds
  • Location icon and location text move to the right by 8 pixels; 0.3 seconds
  • Arrow symbol fades out from 100% opacity to 0% opacity; 0.3 seconds

Feedback
