IDM 241 Microinteractions - Beta

Emanuel

Kitten • Domestic Short Hair 1 mile away

Beta Overview

Triggers

  • Hover over/off image slider dots
  • Click on image slider dots
  • Hover over/off round heart button
  • Click/unclick heart icon

Rules

Hover over image slider dots:
  • Cursor changes to pointer
Hover off image slider dots:
  • Cursor reverts to default
Click on image slider dots:
  • Selected dot changes to a pawprint icon
  • Previously selected pawprint will change to a dot
  • Displayed cat image fades out as new cat image that corresponds to the selected dot fades in; 0.75 seconds
Hover over round heart button:
  • Cursor changes to pointer
  • White circle behind heart icon fades from 50% opacity to 100% opacity; 0.3 seconds
Hover off round heart button:
  • Cursor reverts to default
  • White circle behind heart icon fades from 100% opacity to 50% opacity; 0.3 seconds
Click on heart icon:
  • Purple fades in to fill heart icon; 0.5 seconds
  • Heart icon increases in scale and shrinks back to original size, retaining purple fill; 0.5 seconds
  • The selected cat is added to the user's favorites, which is updated in the database
Unclick heart icon:
  • Purple fill fades out, leaving the heart icon with a stroke and no fill; 0.5 seconds
  • The cat is removed from the user's favorites, which is updated in the database
